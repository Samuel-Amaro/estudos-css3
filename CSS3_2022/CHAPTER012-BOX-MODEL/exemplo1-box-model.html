Demostrando o modelo de caixa(box-model)

O modelo de caixa CSS é essencialmente uma caixa que envolve cada elemento HTML. Ele consiste em: bordas(borders), preenchimento(padding), margens(margins) e o conteúdo real.

Este texto é o conteúdo da caixa. Adicionamos um padding de 50px, uma margin de 20px e uma border verde de 15px.
