A propriedade estilo de borda(border-style)

Esta propriedade especifica que tipo de borda exibir:

Uma borda dotted(pontilhada)

Uma borda dashed(tracejada)

Uma borda solid(solida)

Uma borda double(dupla)

Uma borda groove(sulco)

Uma borda ridge(cume)

Uma borda inset(inserir)

Uma borda outset(inicial)

Uma borda nenhum(none)

Uma borda hidden(oculta)

Uma borda mix(mista)
