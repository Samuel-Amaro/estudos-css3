A propriedade de alinhamento vertical.

Defina o alinhamento vertical de uma imagem em um texto:

vertical-align: baseline(default):

Uma imagen com uma alinhamento padrão

vertical-align: text-rop:

Uma imagem com um alinhamento de texto superior.

vertical-align: text-bottom:

Uma imagem com um alinhamento de fundo de texto.

vertical-align: sub:

Uma imagem com um alinhamento secundário.

vertical-align: sup:

Uma imagem com um super alinhamento.
