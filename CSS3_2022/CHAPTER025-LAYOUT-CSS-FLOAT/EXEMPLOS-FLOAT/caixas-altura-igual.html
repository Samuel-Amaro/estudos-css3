Este exemplo não é muito flexível. Não há problema em usar altura CSS se você puder garantir que as caixas sempre terão a mesma quantidade de conteúdo nelas, mas nem sempre é esse o caso. Se você tentar o exemplo acima em um celular (ou redimensionar a janela do navegador), verá que o conteúdo da segunda caixa será exibido fora da caixa.

Volte ao tutorial e encontre outra solução, se não for isso que você deseja.
