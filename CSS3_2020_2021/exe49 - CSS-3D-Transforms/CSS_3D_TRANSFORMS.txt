# CSS 3D TRANSFORMS

CSS também suporta transformações 3D.

Neste capítulo, você aprenderá sobre a seguinte propriedade CSS:

    transform.

## Métodos de transformação CSS 3D

 Com a transformpropriedade CSS , você pode usar os seguintes métodos de transformação 3D:

    * rotateX()
    * rotateY()
    * rotateZ()
