Melhores Fontes Seguras da Web para HTML e CSS recomendadas pelo W3Scholl

Arial - (Sans-serif)

Arial é a fonte mais usada para mídia online e impressa. Arial também é a fonte padrão no Google Docs.Arial é uma das fontes da web mais seguras e está disponível em todos os principais sistemas operacionais.

Verdana - (sans-serif)

Verdana é uma fonte muito popular. Verdana é facilmente legível, mesmo para tamanhos de fonte pequenos.

Helvetica - (Sans-serif)

A fonte Helvetica é amada por designers. É adequado para muitos tipos de negócios.

Tahoma - (sans-serif)

A fonte Tahoma possui menos espaço entre os caracteres

Trebuchet MS - (Sans-serif)

Trebuchet MS foi desenvolvido pela Microsoft em 1996. Use esta fonte com cuidado. Não é compatível com todos os sistemas operacionais móveis.

Times New Roman - (serif)

Times New Roman é uma das fontes mais conhecidas do mundo. Parece profissional e é usado em muitos jornais e sites de "notícias". É também a fonte primária para dispositivos e aplicativos do Windows.

Geórgia - (Serif)

Georgia é uma elegante fonte com serifa. É muito legível em diferentes tamanhos de fonte, por isso é um bom candidato para design responsivo a dispositivos móveis

Garamond - (Serif)

Garamond é uma fonte clássica usada para muitos livros impressos. Tem uma aparência atemporal e boa legibilidade.

Courier New - (monoespace)

Courier New é a fonte com serifa monoespaçada mais amplamente usada. Courier New é freqüentemente usado com exibições de codificação, e muitos provedores de e-mail o usam como fonte padrão. Courier New também é a fonte padrão para roteiros de filmes.

Brush Script MT - (cursive)

A fonte Brush Script MT foi projetada para imitar a escrita à mão. É elegante e sofisticado, mas pode ser difícil de ler. Use-o com cuidado.
