Modal Imagen

Neste exemplo, usamos CSS para criar um modal (caixa de diálogo) que está oculto por padrão.

Usamos JavaScript para acionar o modal e exibir a imagem atual dentro do modal quando ele é clicado. Observe também que usamos o valor do atributo "alt" da imagem como um texto de legenda da imagem dentro do modal.

Não se preocupe se você não entender o código imediatamente. Quando terminar de usar CSS, vá para nosso Tutorial de JavaScript para saber mais.
