Todas Pseudo-classes

Todas as pseudo-classes CSS

Seletor Exemplo Descrição do Exemplo
:active a : active Seleciona o link ativo
:checked input : checked Seleciona cada <input> element verificado
:disabled input : disabled Seleciona cada elemento <input>
:empty p : empty Seleciona cada elemento <p> que não tem filhos
:enabled input : enabled Seleciona cada elemento <input>
:first-child p : first-child Seleciona todos os <p> elements que é o primeiro filho de seus pais
:first-of-type p : first-of-type Seleciona cada <p> element que é o primeiro <p> element de seu pai
:focus input : focus Seleciona o <input> element que tem foco
:hover a : hover Seleciona links no mouse
:in-range input : in-range Seleciona <input> eles com um valor dentro de uma faixa especificada
:invalid input : invalid Seleciona todos os elementos <input> com um valor inválido
:lang(language) p : lang(it) Seleciona cada <p> element com um valor de atributo lang começando com "ele"
:last-child p : last-child Seleciona cada <p> elements que é o último filho de seu pai
:last-of-type p : last-of-type Seleciona cada <p> element que é o último <p> element de seu pai
:link a : link Seleciona todos os links não visitados
:not(seletor) : not(p) Seleciona cada elemento que não é um elemento <p>
:nth-child(n) p : nth-child(2) Seleciona cada <p> element que é o segundo filho de seu pai
:nth-last-child(n) p : nth-child(2) Seleciona cada <p> element que é o segundo filho de seu pai, contando desde o último filho
:nth-last-of-type(n) p : nth-last-of-type(2) Seleciona cada <p> element que é o segundo <p> element de seu pai, contando a partir do último filho
:nth-of-type(n) p : nth-of-type(2) Seleciona cada <p> element que é o segundo <p> element de seu pai
:only-of-type p : only-of-type Seleciona cada <p> element que é o único <p> element de seu pai
:only-child p : only-child Seleciona cada <p> element que é o único filho de seu pai
:optional input : optional Seleciona <input> eles sem atributo "necessário"
:out-of-range input : out-of-range Seleciona <input> elements com um valor fora de uma faixa especificada
:read-only input : read-only Seleciona <input> eles com um atributo "readonly" especificado
:read-write input : read-write Seleciona <input> elements sem atributo "readonly"
:required input : required Seleciona <input> eles com um atributo "obrigatório" especificado
:root root Seleciona o elemento raiz do documento
:target #news : target Seleciona o elemento ativo #news atual (clicado em uma URL contendo esse nome de âncora)
:valid input : valid Seleciona todos os <input> elementos com um valor válido
:visited a : visited Seleciona todos os links visitados
