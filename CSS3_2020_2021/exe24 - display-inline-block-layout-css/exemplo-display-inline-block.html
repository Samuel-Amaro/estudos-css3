Exemplo de Display: inline-block

Links de Navegação horizontal

Por padrão, os itens da lista são exibidos verticalmente. Neste exemplo, usamos display: inline-block para exibi-los horizontalmente (lado a lado). Nota: Se você redimensionar a janela do navegador, os links serão quebrados automaticamente quando ela ficar muito cheia.
