Clear and Clearfix Layout CSS

Clear(Claro) and Clearfix(Correção Clara) Layout CSS Claro e Correção Clara de Layout CSS

A maneira mais comum de usar a clear propriedade é depois de usar uma float propriedade em um elemento.

Ao limpar os flutuadores, você deve combinar o clear com o flutuante: Se um elemento for flutuado para a esquerda, você deve limpar para a esquerda. Seu elemento flutuado continuará a flutuar, mas o elemento limpo aparecerá abaixo dele na página da web.

O exemplo a seguir limpa o flutuador à esquerda. Significa que nenhum elemento flutuante é permitido no lado esquerdo (do div):

este paragrafo - Observe que este paragrafo é após div1 no código HTML. No entanto, como o div1 flutua para a esquerda, o texto neste paragrafo flui em torno de div1.



Com a propriedade Clear(claro) do Layout CSS

Div1

este paragrado - Aqui, Clear(claro): esquerda(left); move este paragrafo para baixo, abaixo do div2(Conteiner_2) flutuante(float). O valor "left" limpa os elementos flutuados à esquerda. Você também pode limpar "direita" e "ambos os lados".
