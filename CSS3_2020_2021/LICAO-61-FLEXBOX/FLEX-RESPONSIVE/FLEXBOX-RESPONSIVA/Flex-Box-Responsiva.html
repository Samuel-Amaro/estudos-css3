Responsive Flexbox

O "flex-direction: row;" empilha os itens flex horizontalmente (da esquerda para a direita).

O "flex-direction: column;" empilha os itens flexíveis verticalmente (de cima para baixo).

Redimensione a janela do navegador para ver se a direção muda quando o tamanho da tela é de 800 px ou menor.
